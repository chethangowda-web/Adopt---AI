# 🚀 AdoptAI – Autonomous Digital Adoption Agent

> An AI-powered digital assistant that helps users easily access and complete online services such as government applications, banking, healthcare, and educational portals.

## 📌 Problem Statement

Many people struggle to use digital platforms because they are complex, time-consuming, and difficult to understand. This leads to incomplete applications, user frustration, and low digital adoption.

## 💡 Solution

AdoptAI is an intelligent AI assistant that simplifies digital services by:

- 🤖 Understanding user requests in natural language
- 📝 Guiding users through online forms step by step
- 📄 Assisting with document uploads
- ✅ Detecting common mistakes before submission
- 🌐 Providing multilingual support
- ⚡ Making digital services faster and easier

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/adoptai.git
```

### Navigate to the project

```bash
cd adoptai
```

### Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

---

## ▶️ Run the Project

Backend

```bash
npm run dev
```

Frontend

```bash
npm start
```
